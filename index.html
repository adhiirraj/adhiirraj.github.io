Binary Digits

This file doesn't look like much... just a bunch of 1s and 0s. But maybe it's not just random noise. Can you recover anything meaningful from this?

Download the file here.

Solution

We see the head of the file to see that it's a continuous stream of binary digits

% head digits.bin 111111111101100011111111111000000000000000010000010010100100011001001001010001...

We need to split this into 8-bit chunks and then convert binary to decimal

data = open("digits.bin").read() #Loads the file
out = bytes(int(data[i:i+8], 2) for i in range(0, len(data), 8)) #Splits it into 8-bit chunks, then converts binary to decimal
open("output.bin", "wb").write(out) #Writes the output file

Running this script gives us an image

Which is the flag
